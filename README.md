# Breakout - Iván Nicolás Espasandín Suárez
Este repositorio contén un xogo para Linux inspirado no clásico Breakout.

## Instruccións de Uso
Para iniciar o xogo hai que executar o archivo Build.x86_64 que se encontra na raíz do proxecto.  
O xogador controlará unha pala, coas frechas dereita e esquerda.  
Para iniciar o xogo haberá que premer a barra espaciadora. Unha vez iniciado o xogo, para gañar, o xogador debe destruír todos os ladrillos dos dous niveles excepto os ladrillos atravesables que non hai que destruir pero si atravesar como mínimo unha vez.  
O xogador dispón de 5 vidas para acabar o xogo, no caso de perder pódese iniciar unha partida nova premendo calquera tecla.
